  GraalVM
I am fortunate enough to get a chance to work on GraalVM - a high-performance polyglot virtual machine. I work as a Research Assistant (read intern) in the Native Image team , whose goal is to ahead-of-time compile Java code to a standalone executable. My tasks so far have been oriented towards compatibility. It is important to ensure that the semantics of a program remain unchanged or at least as similar as possible.
  Enticing - Semantic Search Engine
As my master thesis project, I created a search engine for querying semantically enhanced documents available within the Knowledge Technology Research Group at FIT BUT. It was a distributed system consisting of multiple types of components. The task also included designing a special query language, it’s compiler and specialized searching algorithms. Backend components were written in Kotlin using Spring. Web frontend was written in TypeScript using React. I successfully designed, developed and deployed the whole system.
  MPS - Meta Programming System
Since my early days at Android, I've been fascinated by IDEs. I was amazed by how much these tools can help you. And it went hand in hand with my interest for programming languages and their analysis. As you can imagine, I naturally gravitated towards JetBrains. So when I got the chance, I went for an internship in the MPS team. MPS is a really cool tool that let's you design your own Domain Specific Languages along with a full IDE support. During my internship, I studied DSL engineering and language engineering in general. In particular, I was extending a tool called Ingrid for converting Antrl4 grammar into MPS project. I designed and implemented an extension to support automatic detection of formatting for a given language.

  SearchBestie
During my bachelor studies, I got involved in the research group VeriFIT, whose focus is automated analysis and verification of systems. There, I maintained the SearchBestie infrastructure for search-based testing of concurrent Java programs using noise injection. Working on this project turned out to be my bachelor thesis. My main task was to integrate SearchBestie with RoadRunner, an open source tool for dynamic analysis of concurrent Java programs. I also designed new heuristics for noise injection and evaluated them. You can find more in the text of the thesis. But be warned that is it in Czech.
